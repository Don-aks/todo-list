# TO-DO List

TO-DO List with smooth animations. Project was created and developed in November 2025 and refactored in June 2026.

![TO-DO List demo](demo.gif)

## Demo

[Demo](https://don-aks.github.io/todo-list/)

## Features

- [x] Add task
- [x] Delete task
- [x] Toggle task completion
- [x] Filter tasks
- [x] Reset filter
- [x] Delete all tasks
- [x] Show tasks count
- [x] Smooth transitions
- [x] Smooth show/hide "list is empty" message animation
- [x] Smooth show/hide "delete all" button animation

## Technologies

- HTML
- CSS
  - Transitions
- JavaScript (Vanilla)
  - Promises
  - ES Modules
  - Async/Await
  - LocalStorage
  - Web Crypto API
